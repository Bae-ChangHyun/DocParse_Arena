# NOTICE

## Inspiration

This project was inspired by:

- **LMSYS Chatbot Arena** (https://chat.lmsys.org/) — the blind-comparison and ELO-based ranking concept
- **OCR Arena** — the idea of applying arena-style evaluation to document parsing
